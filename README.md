<h1 align="center">Jopnal Engine</h1>

### Description

A simple component based 2D/3D game engine.

Jopnal has been tested to work on these operating systems:  
- Windows 7 & 10 (VC++ 12/13)
- Android 4.0.3< (NDK 10e through 12b)
